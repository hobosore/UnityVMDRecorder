# UnityVMDRecorder

## About
Unity上の人・カメラアニメーションをランタイムで記録してvmd(MMDのアニメーションファイル)に保存するためのコードです。  
This code enables you to record humanoid/camera animations in unity into .vmd file (MikuMikuDance).
